# ADR-002: Confirm Render as Hosting Platform

**Status:** Accepted  
**Date:** 2026-04-19  
**Relates to:** ADR-001

## Context
The project brief in Section 3.3 specifies Azure as the required hosting platform. Our group attempted Azure setup and encountered blocking issues that prevented progress. We raised this with the course lecturer, who authorised us to proceed with Render for this project.

## Decision
KnockKnock.prof will be deployed on Render, as originally proposed in ADR-001. This decision is confirmed with lecturer authorisation.

## Consequences
- Deployment is unblocked, so the team can continue with implementation.
- A record of the lecturer's authorisation should be kept in case it is requested during marking.
- Render's GitHub integration simplifies deployment and supports faster team progress.
